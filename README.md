## 📖 Overview

<a href="#endNote">EndNote</a> serves academic researchers, graduate students, scientists, medical professionals, and anyone who works with research literature — building literature review collections, writing manuscripts that require proper citations, and managing the growing reference library that ongoing research produces.

The <a href="#endNote">literature review use case</a> collects all relevant papers on a research topic from multiple databases into an organized EndNote library — conducting the systematic search across PubMed, Web of Science, and other sources and organizing the complete reference set for review without manual citation tracking. The <a href="#endNote">manuscript writing use case</a> uses Cite While You Write throughout the drafting process — adding citations as evidence is introduced in the text and generating the complete formatted bibliography when the manuscript is complete.

The <a href="#endNote">graduate student use case</a> manages the growing reference library of literature across coursework, qualifying examinations, and dissertation research — years of reference collection organized and searchable, with citations always available for paper writing. <a href="#endNote">Clinical research use case</a> in medical and health sciences collects clinical trial literature, systematic review sources, and evidence-based medicine references in an organized library supporting clinical research and evidence synthesis.</p>

<a href="#endNote">Citation style customization</a> modifies existing citation styles or creates new ones for journals with unique citation requirements — the style editor handles the specific formatting rules that any publication requires. <a href="#endNote">Duplicate detection and removal</a> identifies duplicate references in the library — when importing from multiple databases that index the same papers, duplicates appear and are removable to keep the library clean.

<a href="#endNote">Subscription and perpetual license options</a> provide different access models — Clarivate offers both ongoing subscription access and perpetual license options. <a href="#endNote">Clarivate development</a> as part of the Web of Science product family integrates EndNoteith the research information infrastructure that universities and research institutions use for discovery and citation tracking.

The 7000-plus citation style library reflects the genuine diversity of citation requirements across academic disciplines and publications. Different academic fields use incompatible citation systems — medical journals typically use Vancouver numbered references, social science journals use APA author-date, humanities journals use Chicago or MLA. Within each field, individual journals often have specific formatting variations. EndNote's style library covers these differences, making manuscript submission to specific journals practical without reformatting every citation manually.

## 💬 FAQ

<details>
<summary>What are Smart Groups in EndNote?</summary>

Smart Groups automatically include references matching configured search criteria — keywords, authors, years, or other fields — updating dynamically as new matching references enter the library.

</details>

<details>
<summary>Can EndNote detect duplicate references?</summary>

Yes. Duplicate detection identifies references appearing multiple times in the library, which commonly occurs when importing from multiple databases indexing the same papers.

</details>

<details>
<summary>Can I customize citation styles in EndNote?</summary>

Yes. The style editor modifies existing styles or creates new citation formats for journals with specific requirements.

</details>

<details>
<summary>Does EndNote support systematic reviews?</summary>

Yes. Literature search import, organization, and complete reference management serve systematic review workflows.

</details>

<details>
<summary>What is EndNote Online?</summary>

EndNote Online is the web-based component of EndNote that syncs with the desktop application for library access from any browser and additional Mac installations.

</details>
